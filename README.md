# David Cut Tracker

Free phone-ready college cut tracker for daily logs, cheap high-protein meals, food budget, freezer packs, lifting, walking, and cheat/social-day guardrails.

Live app URL:

https://com03.github.io/david-cut-tracker/

The same tracker files are also on the `gh-pages` branch so GitHub Pages can publish from either the GitHub Actions workflow or the classic branch source.

Your logs are saved in your browser on the device you use. On iPhone, open the live app in Safari, then use Share -> Add to Home Screen. Use Export Backup regularly.
